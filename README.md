# Data Science & Analytics Studies 📚

This repository serves as my personal learning library. It contains daily exercises, bootcamp materials, and technical deep-dives into the data science ecosystem.

## Repository Structure

- **/python-logic:** Pure logic exercises and algorithmic thinking with Python.
- **/sql-mastery:** SQL scripts, database modeling practices.
- **/generation-bootcamp:** Projects and assignments developed during the Generation Brazil Data Analytics Bootcamp.
- **/colab-archives:** Research notebooks and library explorations (Pandas, NumPy, Matplotlib, Seaborn).

## Main Libraries Explored
- **Data Manipulation:** Pandas, NumPy.
- **Visualization:** Matplotlib, Seaborn.
- **Environment:** Google Colab, Jupyter Notebooks.
